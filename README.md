## ğŸ’¡ Object-Oriented Programming (OOPs)

**Object-Oriented Programming (OOPs)** is a core paradigm in C++ that structures programs around **objects**â€”real-world entities that encapsulate both **data** and **behavior**. It allows for more modular, reusable, and maintainable code.

OOP in C++ is built on four fundamental principles:

* **Encapsulation:** Bundling of data and methods that operate on that data, while restricting direct access to internal details.
* **Abstraction:** Hiding the complex internal workings of classes and exposing only the necessary parts of functionality.
* **Inheritance:** Enabling one class to derive properties and behaviors from another, promoting code reuse and hierarchy.
* **Polymorphism:** Allowing functions and objects to take on multiple forms, enhancing flexibility and dynamic behavior in programs.
